✅ ¿Qué es GitHub Pages?

GitHub Pages es un servicio de alojamiento web gratuito que te permite publicar páginas web directamente desde un repositorio de GitHub. Ideal para páginas personales, de negocios, portafolios o landing pages.

⚙️ ¿Cómo funciona?

GitHub Pages toma los archivos HTML, CSS y JavaScript que subes a un repositorio público, y los publica automáticamente como una web accesible al mundo. Tú no pagas hosting.

Ejemplo:
Si tu usuario es rafogaudi, tu sitio puede verse así:
👉 https://rafogaudi.github.io

🚀 Pasos para crear tu web con GitHub Pages

1. Crea una cuenta en GitHub

  • Ve a github.com y regístrate.

  • Elige un nombre de usuario corto y profesional (ej: gaudiperu, minzenails).

2. Crea un nuevo repositorio

  • Dale clic a "New repository".

  • Nombre del repo: gaudiperu.github.io (¡debe tener exactamente ese formato!).

  • Marca la opción de "Public".

  • Inicialízalo con un README si quieres.

3. Sube tu página web

Tienes dos opciones:

Opción A – Sitio básico (sin programar):

  • Crea un archivo index.html (puede ser un diseño hecho en Canva, Figma, o plantillas gratuitas).

  • Sube ese archivo al repositorio (drag & drop si quieres).

  • ¡Listo! Tu página estará publicada en:
    👉 https://gaudiperu.github.io

Opción B – Con editor como Visual Studio Code:

  • Diseñas tu sitio con HTML/CSS/JS localmente.

  • Subes todo el proyecto al repo con Git o vía GitHub Desktop.

4. (Opcional) Personaliza el dominio

  • Puedes comprar un dominio como www.gaudiperu.com.

  • Configuras DNS para apuntar a GitHub Pages (yo te puedo guiar con eso si decides hacerlo).

📌 ¿Qué puedes poner en tu sitio?

  • Catálogo de cursos.

  • Formulario de contacto.

  • Botón de WhatsApp.

  • Mapa de ubicación.

  • Reseñas y fotos.

  • Enlace a pagos (Yape, Plin, etc.).
